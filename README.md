🚴 Bike Sales Dashboard (Excel)
📌 Project Overview

This project is an interactive Bike Sales Dashboard created in Microsoft Excel to analyze customer demographics and purchasing patterns. The dashboard helps identify factors influencing bike purchases and provides insights through visualizations and filters.

🎯 Objectives
Analyze customer data related to bike purchases.
Identify trends based on age, income, gender, occupation, and region.
Create an interactive dashboard for quick decision-making.
Demonstrate Excel data analysis and visualization skills.

🛠 Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Conditional Formatting
- Dashboard Design

📊 Dashboard Insights
- Comparison of bike purchases by gender.
- Relationship between customer income and bike purchases.
- Analysis of purchases across different age groups.
- Regional comparison of bike sales.
- Customer segmentation based on marital status and occupation.
📂 Dataset

The dataset contains customer demographic information including:

Age
Gender
Income
Occupation
Marital Status
Region
Bike Purchase Status
